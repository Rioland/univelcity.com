]]> About The Course

Learn to build dynamic website interfaces with HTML, CSS, Bootstrap Javascript and React with some new features like SAAS.
Learn how to prototype design in Figma and how to convert it to HTML and CSS.
Learn to incorporate custom fonts, font awesome icons, and chrome developer tools for your applications
Learn to structure your pages using Bootstrap grid system
Understand how to use card interface to recreate interfaces
Understand how to configure and get started with your own customized version of Bootstrap
Work with modular and customized blocks of content called “cards” to display various text and images
Design responsive and accessible tables, and create a refined and accessible user experience
Explore third-party integration to extend Bootstrap’s component and visual structure through icons to complex form validation.
Build a dynamic website using ES6 and React, And how to consume API’s with React.

 

Course Outline

Week 1: Introduction to Computer Science and Web Tools(Github and Command Lines)

Week 2: HTML and HTML5

Week 3: CSS and CSS3

Week 4: Responsive Design

Week 5: Bootstrap 4

Week 6: Introduction to Javascript

Week 7: Mid-Term Projects & Assessments

Week 8: JQuery

Week 9: ES6

Week 10: React 1: Getting Started With ReactJS

Week 11: React 2: Working With Data

Week 12: Final Projects Presentation & Graduation
